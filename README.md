# README

## CommanderGAI

### Overview

**Note** The project is in progress. Nothing done yet.

CommanderGAI is an experimental project whose goal is to explore building a useful and practical software program where the source code itself is generated by Generative AI using prompts.

Instead of manually writing most of the implementation, the project relies on AI-assisted generation to create and evolve the codebase. The intention is to demonstrate a workflow where prompts drive development and the resulting generated source is compiled and run using standard development tools, just like a typical software project written by developers.

The generated program, **cgai**, is a **file commander tool** inspired by classic file managers such as **Norton Commander**. The goal is to create a practical command-line driven file management utility that works across modern operating systems.

The program is designed to run on the major desktop platforms:

* Windows
* Linux
* macOS

### How It Works

The project uses the ChatGPT API to generate source code based on prompts defined in the repository.

A Python script orchestrates the generation process, retrieves the generated code, and prepares the project for compilation.

### Generating and Building the Project

To generate and build the source code, run the generation script:

`python generate.py`

The script will perform the following steps:

1. Call the ChatGPT API to generate the required source code.
2. Create or update the source files.
3. Configure the build system.
4. Build the generated program.

### Requirements

The following tools must be installed before running the project:

* Python 3
* CMake
* Ninja
* A C/C++ compiler such as:

  * Visual Studio (MSVC)
  * Clang
  * GCC
  * Xcode (Apple toolchain)

Additionally, ChatGPT API access must be configured so that the generation script can request code generation.
